TaxiCase

This project aims to acquire, store, and make available the NYC Taxi data from January through May 2023. The data is sourced directly from the NYC Taxi & Limousine Commission (TLC) website. This case study constitutes a mandatory part of the iFood Data Engineer recruitment process.

Read More

Solution

DataLake





The pipeline reads the original data, ingesting it into the Data Lake. The raw data is then stored in the Bronze layer as a Delta table, and the aggregated, cleansed, and normalized results are persisted in the Silver layer as a Delta Live Tables asset.

Analysis





With the data now available in the Data Lake, analytical SQL queries were developed to address the case's key questions, which include:

- The average total amount earned per month, focusing exclusively on Yellow Taxis.

- The average passenger count per hour of the day for rides taken in May, considering all taxi types (Yellow and Green).

DataViz





The final stage involved creating a Databricks Dashboard and securely embedding it within this website. The entire solution is hosted using a robust, serverless architecture leveraging AWS CloudFront, Amazon S3, AWS Certificate Manager (ACM), and Amazon Route 53.
